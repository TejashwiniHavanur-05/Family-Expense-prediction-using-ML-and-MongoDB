# 📊 Family Expense Prediction using Machine Learning

## 📌 Project Overview
This mini project is designed to estimate future family expenses using Machine Learning.  
The system is trained on previous expense data and provides predictions based on user inputs.

---

## 🚀 Technologies Used
- Python  
- Machine Learning Algorithms  
- Flask (Web Framework)  
- MongoDB (Database)  
- HTML  
- CSV & JSON  

---

## 📁 Project Files
- `app.py` – Flask application for handling requests and responses  
- `train_model.py` – Model training using CSV dataset  
- `train_model_mongodb.py` – Model training using MongoDB data  
- `ml_model.py` – Contains core ML logic  
- `convert_csv_to_json.py` – Converts CSV file into JSON format  
- `mongodb_insert.py` – Inserts data into MongoDB database  
- `expenses.csv` – Raw dataset  
- `expenses.json` – Converted dataset  
- `model.pkl` – Saved trained model  
- `index.html` – Frontend interface  
- `famlreport1.pdf` – Detailed project report  

---

## ⚙️ System Workflow
1. Collect and preprocess expense data  
2. Train the machine learning model  
3. Save the trained model for reuse  
4. Accept user input through the web interface  
5. Predict expenses using the trained model  
6. Store and manage data using MongoDB  

---

## 📈 Results
- Generates predicted expense values  
- Displays output on the web interface  
- Stores input and results in MongoDB
